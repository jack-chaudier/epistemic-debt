# Evidence ledger

Evidence classes: `THEOREM` proved · `EXACT` exhaustive computation · `PREREGISTERED` predictions fixed before run · `OBSERVED` pattern, not confirmatory · `REFUTED` · `CONJECTURE`. Outcome modifiers: `VOID` = frozen prerequisite failed, reuse only as exploratory/`OBSERVED`; `LATENT` = capability crossed a frozen recovery threshold; `INAPPLICABLE` = a guard blocks the frozen headline; `SPLIT` = panel or conjuncts disagree. Modifiers never upgrade the evidence class.
